# AudioVisualizer2.0
---

### A simple (old) JavaFX-based AudioVisualizer

***Supports: Windows, Linux***

### Linux Requirements

For mp3 codec support, install `libavcodec-extra`. 
```bash
# eg. on Debian
sudo apt-get install libavcodec-extra
```

Alternatively, for further media type support, install your flavor of `restricted-extras`

```bash
# eg. on Ubuntu
sudo apt-get install ubuntu-restricted-extras
```
